REQUIREMENTS:
- Python 3.x
- Full Project Directory
-------------------------------------

HOW TO RUN:
- Configure the server (Optional) 
	+ Open config.json
	+ Adjust configurations and save
- Run 'server.py'
-------------------------------------

HOW TO TEST:
- Open a Web browser
- Replace 'ServerIP' with your device IP and visit the following URLs
	+ http://ServerIP:6789/                      -> directs to HelloWorld default page
	+ http://ServerIP:6789/status                -> directs to server status page
	+ http://ServerIP:6789/invalid-file.html     -> directs to 404 page
	+ http://ServerIP:6789/image.png             -> directs to image
